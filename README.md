# The Odin Project - Fundamentals > Javascript project
##Rock Paper Scissors
This is a small javascript functional programming project. 
 - As per the classic RPS game, the user selects and option.
 - The computer does the same
 - Outcomes are as follows
 - - Rock beats Scissors
   - Paper beats Rock
   - Scissors beats Paper
 - The game is played a total of 5 times
 - The final score is then displayed
